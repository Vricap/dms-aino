# Document Management System  

Platform manajemen dokumen PT. AINO yang memungkinkan pembuatan, penandatanganan, dan pelacakan dokumnn secara digital, dengan dukungan sequential, parallel, dan group signing.  

Coba: https://dms-aino.onrender.com/  
Lihat screenshot: https://vricap.github.io/portfolio/screenshot/dms/screenshot.html  

Tech Stack:
* React for frontend
* NodeJS for backend
* Mongodb with Mongoose for database
* JWT for authentication
