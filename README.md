# Document Management System  

Platform manajemen dokumen yang memungkinkan upload, lihat, kirim, tanda tangan, audit dan hapus dokumen secara digital, dengan dukungan sequential, parallel, dan group signing.  

Coba: https://dms-aino.onrender.com/  
Lihat screenshot: https://vricap.github.io/portfolio/screenshot/dms/screenshot.html  

Tech Stack:
* React untuk frontend
* NodeJS untuk backend
* Mongodb dengan Mongoose untuk database
* JWT untuk autentikasi
